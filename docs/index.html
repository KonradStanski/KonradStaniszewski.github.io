Open Air Laser Communication

  • Designed an open air laser communication system that allows two way communication of text.
  • Writen in C++ and uses a class structure for the Laser and Sensor providing a high level API for sending and receiving messages.
  • Employed a number of error correction codes such as the hamming (7,4) code, and checksums to ensure the quality of transmitted data.

  • Used a TCP like SYNACK client-server topology for transmission of messages and making sure that they arrived correctly.
  • Capable of up to 2 kb/s data transmisison speeds.
